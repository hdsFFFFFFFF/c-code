#### 64位编程模型

64位`Linux`系统采用`LP64`编程模型。

| Type           | size(Byte) |
| -------------- | ---------- |
| char           | 1          |
| unsigned char  | 1          |
| short          | 2          |
| unsigned short | 2          |
| int            | 4          |
| unsigned int   | 4          |
| long           | 8          |
| unsigned long  | 8          |
| float          | 4          |
| double         | 8          |
| long int       | 8          |
| long long      | 8          |
| long double    | 16         |
| size_t         | 8          |
